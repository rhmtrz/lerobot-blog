> 本記事は LeRobot チュートリアルシリーズのハードウェア前提です。LeRobot をインストールする前に、この記事の作業を終わらせてください。

## はじめに

この記事では、SO-100 のキットから 2 本のアーム — **Leader**(テレオペ時に人間が手で動かす側)と **Follower**(タスクを実際に実行する側)— を組み立てる手順を解説します。最終的に、両アームの組み立て・配線・USB 認識まで完了し、LeRobot シリーズの残りの記事でそのまま使える状態になります。想定読者は、ドライバーとターミナルの基本操作はわかるけれど、ロボットアームの組み立ては初めて、という方です。

Leader と Follower の役割やテレオペの流れについては [プライマー](./lerobot-primer-and-tips) を参照してください。本記事はハードウェアの組み立てのみを扱います。

## ゴール

この記事を読み終えると、以下の状態になっています。

- ギアを取り外した Leader アーム(人間が低摩擦で動かせる状態)
- 電動グリッパー付きの Follower アーム
- 各アームの STS3215 モーター 6 個に一意な ID(`1`〜`6`)が書き込まれ、デイジーチェーンで接続されている
- 両アームが USB で認識され、`lerobot-setup-motors` から応答する

キット 1 本あたりの費用は **約 $123(2 本で $246)**。これに加えてフィラメント代と、1 アームあたり 6 時間ほどの印刷時間がかかります。

## 前提

### 工具

- 小さいマイナスドライバー(3D プリントのサポート除去用)
- 精密プラスドライバーセット
- 六角レンチセット — `<TBD: SO-100 で使うサイズを確認>`
- 3D プリンター(または STL を受け付ける印刷代行サービス)
- (オプション)バイス・クランプを 2〜4 個。肩関節や肘関節をネジ止めするときに手で押さえ続けるのは大変で、1 台目はクランプがあるとストレスが大幅に減ります。
- (オプション)ESD マットと、ネジを入れる小さなトレイ

### 3D プリント部品

STL ファイルはすべて [SO-ARM100 リポジトリ](https://github.com/TheRobotStudio/SO-ARM100) にあります。公式の `SO100.md` に記載されている印刷設定は次のとおりです。

| 設定項目 | 値 |
|----------|-----|
| 素材 | PLA / PLA+ |
| ノズル径 | 0.4 mm(または 0.6 mm) |
| レイヤー高 | 0.2 mm(0.6 mm ノズルなら 0.4 mm) |
| インフィル | 13〜15 % |
| サポート | あり。Ender 3 と Prusa MINI+ 用の向き指定ファイル付き |

1 アームにつき 1 セット印刷します。1 セットあたり `<TBD: スライサーで確認した総フィラメント量・時間>` ほどです。

:::message
組み立てる前にサポート材を **完全に** 除去してください。モーターブラケットやホーンの差し込み部はそもそも公差がタイトで、サポート材が少しでも残っていると、5 分で終わる工程が 30 分かかるようになります。
:::

### 作業環境

- 平らで明るい作業スペース
- 小さなネジ用のトレイか磁石トレイ(必ず落とします)
- モーター ID を書き込む工程用に、USB-C ケーブルと macOS / Ubuntu のホスト PC

## 組み立て手順

### 1. パーツの印刷とサポート除去

上記の設定で STL をスライスし、1 セット印刷します。冷めたら、小さなマイナスドライバーでサポート材を取り除きます。組み立て前に各モーターをブラケットへ仮組みしてみてください。すっと入らない場合は、内壁をヤスリやカッターで整えます。**この作業は今のうちに**、ブラケットが単体の状態で済ませてください。アームに取り付けた後では大変です。

### 2. 機械組み立ての *前に* モーター ID を設定する

本記事で最も重要な工程です。STS3215 はすべて出荷時 ID が `1` ですが、アームのデイジーチェーンには `1`〜`6`(根元 → グリッパー)の一意な ID が必要です。さらに重要なのは、組み立て後はモーターのコネクタがブラケット内部に隠れてしまう点です。ID 設定をスキップして先に組み立てると、修正のためにアームを部分的に分解することになります。

Waveshare 基板をホスト PC に USB-C で接続し、ジャンパーを **B** 側に設定したうえで、モーターを **1 個ずつ** 順に接続します。

シリアルポートを確認します。

```bash:terminal
lerobot-find-port
```

基板を接続したときに現れ、抜くと消えるポートが対象です。macOS では通常 `/dev/tty.usbmodem...`、Linux では `/dev/ttyACM0` または `/dev/ttyUSB0` として見えます。

次に、モーターを 1 個ずつ接続しながら ID を書き込みます。

```bash:terminal
lerobot-setup-motors \
    --robot.type=so100_follower \
    --robot.port=/dev/ttyACM0
```

画面の指示に従って、順番にモーターを接続していきます(1 = 根元の肩パン、6 = グリッパー)。Leader 側も同じ手順を `--robot.type=so100_leader` で繰り返します。

:::message
**Linux のみ**: シリアルポートを開く際に `Permission denied` が出る場合は、`dialout` グループに自分を追加(`sudo usermod -aG dialout $USER` を実行後、ログインしなおし)するか、現在のセッション限定で `sudo chmod 666 /dev/ttyACM0` を実行してください。
:::

:::message alert
この工程を飛ばして「組み立てた後で対応する」のはやめてください。組み込んだ後にモーター ID を書き換えるには、そのモーターを固定しているブラケットを外す必要があります。すべてのモーターが手元で自由に扱えるうちに ID を設定してください。
:::

### 3. ベースと肩パン(モーター 1)の組み立て

- モーター 1 をベースのモーターブラケットにはめ込みます。
- `<TBD: ネジ本数>` 本のネジでモーターをブラケットに固定します。
- モーターホーンを出力軸に取り付けます。
- ブラケットを印刷したベースプレートにネジ止めします。

ここがアームの土台になり、残りの関節はすべてここから片持ち梁の形でぶら下がります。「後でまとめて締めればいい」と思ってベースプレートのネジを飛ばさないでください。関節を 1 つ追加するたびに、アームをひっくり返すのが難しくなります。

### 4. 肩ロール(モーター 2)と上腕の取り付け

- ネジ止めする前に、モーター 1 のケーブルをモーター 2 のブラケットの溝に **先に通しておきます**。
- モーター 2 をブラケットに取り付け、ホーンを装着し、肩パンの出力軸に固定します。
- 上腕の構造パーツを `<TBD: 片側のネジ本数>` 本のネジでモーター 2 のホーンに取り付けます。

「先にケーブルを通す」という工程は、ここから先のすべての関節で繰り返します。何もないブラケットにケーブルを通すほうが、ブラケットが挟まれた状態で後からケーブルを通すよりはるかに速く済みます。

### 5. 肘(モーター 3)と前腕の取り付け

- モーター 2 のケーブルを先に通します。
- モーター 3 をブラケットに取り付け、ホーンを装着し、上腕の出力軸にネジ止めします。
- 前腕のパーツを取り付けます。

### 6. 手首(モーター 4・5)の取り付け

- モーター 4 は手首のピッチ(縦回転)です。前腕の出力軸に取り付けます。
- モーター 5 は手首のロール(横回転)です。モーター 4 の出力軸に取り付けます。ロール側はモーターホーンを 1 個だけ使います。
- 両方のケーブルを先に通しておきます。

### 7. エンドエフェクター — Follower / Leader

この工程は、組み立てているアームによって分岐します。先に Follower から組むのがおすすめです。Leader より単純です。

#### 7a. Follower: グリッパー

- モーター 6 をグリッパーホルダーに取り付けます。
- `<TBD: ネジ本数>` 本のネジでグリッパーの爪をモーター 6 のホーンに取り付けます。
- モーター 5 → モーター 6 の 3 ピンケーブルを接続します。

Follower のモーター 6 は通常の電動サーボです。Policy が指令したときにグリッパーを開閉します。

#### 7b. Leader: ギア除去とトリガーハンドル

Leader は **受動的** なアームで、人間が手で動かします。自然に動かせるようにするため、各 STS3215 の内部ギアを取り外して、モーターを自由に回せる(位置だけを読み取る)状態にします。ギアの取り外し手順は [Hugging Face の SO-100 ドキュメント](https://huggingface.co/docs/lerobot/en/so100) からリンクされている動画を参照してください。きれいに外すには少し練習が必要です。

:::message alert
ギアの取り外しは予備パーツがない限り基本的に不可逆で、取り外した後のモーターは電動サーボとしては動作しません。Follower 用に使うモーターのギアは絶対に取り外さないでください。Leader 用の 6 個のモーターは、Leader の機械組み立てに入る **前に** すべてギア除去を済ませてください。
:::

ギアの取り外しが終わったら、

- 手首側に Leader 用のホルダーをモーター 5 に取り付けます(グリッパーの爪は付けません)。
- `<TBD: ネジ本数>` 本のネジでトリガーハンドルをホルダーに取り付けます。
- モーター 6 をトリガー部に取り付け、トリガーレバーをホーンに装着します。

Leader のモーター 6 もギアなしです。トリガーの位置を読み取って、Follower のグリッパーに反映させる役割を担います。

### 8. コントローラーの取り付けとモーターのデイジーチェーン

- `<TBD: ネジ本数>` 本のネジで Waveshare 基板をベース背面に固定します。
- 3 ピンケーブルを基板 → モーター 1 → モーター 2 → … → モーター 6 の順でデイジーチェーン接続します。
- Waveshare 基板に電源を接続します。

:::message alert
モーターケーブルが 1 本でも外れた状態、またはジャンパーが誤った位置のまま電源を入れないでください。電源投入時にコネクタが緩んでいると、モーターが未定義の状態になり、ID の再書き込みが必要になることがあります。
:::

### 9. 電源投入とスモークテスト

USB ケーブルと電源を接続し、OS からコントローラーが見えていることを確認します。

```bash:terminal
# macOS
ls /dev/tty.usbmodem*

# Linux
ls /dev/ttyACM* /dev/ttyUSB*
```

接続したコントローラーの数だけ、新しいポートが見えるはずです。続いて、6 個のモーターすべてが応答することを確認します。

```bash:terminal
lerobot-setup-motors \
    --robot.type=so100_follower \
    --robot.port=/dev/ttyACM0
```

スクリプトが `Connected to 6 motors` と表示すれば、キャリブレーションに進める状態です。Leader 側も同じ手順を行います。

## つまずきポイント

**よくあるエラー**: `Connected to N motors` で N が 6 未満になる
**原因**: デイジーチェーンのどこかで 3 ピンケーブルが断線・未挿入、もしくはモーター ID が設定されていない。
**対処**: 電源を切り、基板側から順にすべてのケーブルを挿し直します。改善しない場合は ID の衝突を疑い、モーターを 1 個ずつ外して欠番や重複を特定します。最悪の場合、一部を分解して該当モーターの ID を再書き込みします。

**症状**: 組み立て後にモーターがまったく検出されない
**原因**: ID が未設定、もしくは Leader 用に書き込んだモーターを Follower に取り付けてしまった(またはその逆)。
**対処**: 該当モーターのブラケットを外し、正しい `--robot.type` で `lerobot-setup-motors` を実行してから再度組み付けます。

**症状**: Leader の関節が固く、動かすのに力がいる
**原因**: 1 個以上のモーターでギアが完全に取り外されていない。
**対処**: 固い関節を触って特定し、そのモーターを外してギア除去を完了させ、再度取り付けます。

**よくあるエラー**: `Could not open port /dev/ttyACM0`(Linux)
**原因**: 自分のユーザーが `dialout` グループに入っていないか、別のプロセスがポートを掴んでいる。
**対処**: `sudo usermod -aG dialout $USER` を実行してログインしなおすか、現在のセッション限定で `sudo chmod 666 /dev/ttyACM0` を実行します。

**症状**: 1 台目に 3 時間かかってヘトヘト
**原因**: 正常です。公式ドキュメントでも、1 台目は 1 時間以上、2 台目はそれより大幅に短くなると書かれています。
**対処**: 2 台目に取りかかる前に必ず休憩してください。1 日で両方仕上げようとするのは禁物です。疲労状態でのギア除去ミスは最悪の結果につながります。

## まとめ

- Leader と Follower の 2 アームを、それぞれ 1 セットの BOM(約 $123/アーム)で組み立てた。
- 各アームの STS3215 モーター 6 個に、機械組み立ての **前に** 一意な ID `1`〜`6` を書き込んだ。
- Leader のモーターはギアを取り外して人間が自由に動かせる状態にし、Follower はギアを残して通常動作させる。
- Waveshare コントローラーをデイジーチェーンで配線し、ジャンパーを **B** に設定。ホスト側で USB として認識される状態にした。
- 両アームで `lerobot-setup-motors` を実行し、`Connected to 6 motors` を確認した。
